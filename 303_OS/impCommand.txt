✅ Why explorer . Doesn't Work in WSL
In PowerShell or Command Prompt, explorer . opens the current folder in Windows File Explorer — because explorer.exe is a NATIVE WINDOWS command.





1. safe DELETE

rm -i filename
rm -ri foldername


2. 🧾 VIEW FILE CONTENTS
------------------------------------
# View file line by line
cat fileName.txt

# View file page by page (with scroll)
less fileName.txt

# View beginning lines
head -n 10 fileName.txt

# View last lines
tail -n 10 fileName.txt



3. 📂 LIST FILES IN DIRECTORY
------------------------------------
# List files and folders
ls

# List all, including hidden files
ls -a

# List with detailed info (permissions, size, date)
ls -l

# Combined
ls -la



4. 📦 MOVE / RENAME FILE OR FOLDER
------------------------------------
# Rename a file
mv oldName.txt newName.txt

# Move a file to another folder
mv fileName.txt /path/to/destination/


5. 📄 COPY FILE OR FOLDER
------------------------------------
# Copy a file
cp source.txt destination.txt

# Copy folder and contents
cp -r sourceFolder/ destinationFolder/






6. 🔍 SEARCH
------------------------------------
# Search for a file in current directory
find . -name "fileName.txt"

# Search text inside a file
grep "textToFind" fileName.txt


7. 📅 DATE & TIME
------------------------------------
# Show current date and time
date


------------------------------------
📊 DISK USAGE
------------------------------------
# Show disk usage of current folder (summary)
du -sh .



# Show free disk space
df -h


------------------------------------
 👤 USER INFO
------------------------------------
whoami     # Show current user
id         # Show user ID and group info



8. 🔒 PERMISSIONS
------------------------------------
# Make script executable
chmod +x script.sh

# View file permissions
ls -l



9. 🧠 SYSTEM INFO
------------------------------------
# Show OS and kernel
uname -a

# Show CPU info
lscpu

# Show memory info
free -h

# Show running processes
top

•	To get a report on the system’s disk space usage
$ df	# report in % and KBs
$ df –m	# report in Megabytes


•	To check how much space a file or a directory has (File size)
$ du filename	# check disk space of a file or directory
$ du ~/filename	# file size in KBs in a given directory
$ du –sh ~/filename	# File or Folder size of a given directory
$ du –ah ~/filename	# shows disk space for all files/directory
